A simple Git flow that works for Agile development

![image.png](/assets/image_1670212018129_0.png)

The flow:

- Creating a new release branch for every sprint.

- Merge all the development branches into the sprint release branch, then do the tests.

- Merge the branches which passed the tests into the main branch.

- Remove and create a new one for the next sprint.
